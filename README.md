# To-do list
This is a To-do list [website](https://https://todo-wind.netlify.app/to-do-list) made using reactjs.


## About
  You can create, delete, rename tasks.
  
Features 
 * Double click the checkbox to delete the task 
 * others features are intuitive 

## Packages used

* @chakra-ui/react 
* @emotion/react
* @emotion/styled 
* framer-motion
* react-icons 
* react-router-dom
* vite
